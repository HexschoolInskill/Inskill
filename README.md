# Inskill 多元內容創作課程平台

## VSCode 插件建議

- EditorConfig for VS Code
- Prettier(程式碼格式化工具)
- Tailwind CSS IntelliSense(TailwindCSS 提示工具)
- Volar(不推薦使用 Vetur)

## 初始化專案

```bash
yarn or npm init
```

## Commit 提交方式(重要)

```bash
yarn commit or npm run commit
```

Commit 類型參考:
https://hackmd.io/@Heidi-Liu/git-commit-message
